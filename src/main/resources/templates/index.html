IIIF Server Demo

This is a standalone demo IIIF server demonstrating the usage and features of IIIF Image API Java Library and IIIF Presentation API Java Library (Java implementations of the IIIF specifications) and showcasing different IIIF viewers. You can have a look at its code as reference implementation for using the IIIF API Java Libraries.

  • The IIIF Image API for accessing images provided by an IIIF server is specified at http://iiif.io/api/image/2.1/.
  • The main objective of the IIIF Presentation API (specified at http://iiif.io/api/presentation/2.1/) is to provide the information necessary to allow a rich, online viewing environment for primarily image-based objects to be presented to a human user.
